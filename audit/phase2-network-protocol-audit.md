## Phase 2A Summary

Numeric protocol, CBOR framing/encoding, compression, RSA Setup handshake, reply/notify shapes, and room-list/heartbeat protocol are faithfully restored through the gateway+protocol packages. The browser transport is a deliberate WebSocket-gateway adapter (one WS ↔ one asio TCP).

**Key changes since the M2-era report:**
- P2A-009 MD5 → **successfully restored** (M5-c byte-exact `computeFlistMd5`, verified live against asio for both the core-only and +3-pack sets, and that a wrong MD5 is rejected).
- P2A-014 Heartbeat → **successfully restored** (M3+ lobby-phase echo).
- New rows P2A-020/021/022/023 for Observe/Reconnect bootstrap, reconnect credentials, and lobby command handlers (M3+).

Remaining not-restored / simplified: UDP LAN discovery, detailed login-error surface + UpdatePackage download UI, full `expectedReplyIds` set, `server_lag` computation, and `SetServerSettings`/`AddTotalGameTime` consumption.
